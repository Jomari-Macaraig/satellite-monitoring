# Setting up development
### 1. Cloning the project.
```commandline
$ git clone <repo-url> satellite-monitoring
$ cd satellite-monitoring
```

### 2. Create virtual environment
```commandline
$ virtualenv venv
```

### 3. Install requirements
```commandline
$ pip install -r requirements/dev.pip
```
